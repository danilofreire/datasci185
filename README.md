# DATASCI 185 - Introduction to AI Applications

This repository contains materials for the course [DATASCI 185 - Introduction to AI Applications](https://danilofreire.github.io/datasci185) at [Emory University](http://www.emory.edu). The course offers a non-technical introduction to artificial intelligence and its effects on institutions, work and everyday life.

## Repository Structure

This repository is organised with two main branches:

### Main Branch
Contains the source files for the course:
- `syllabus.qmd` - Course syllabus in Quarto format
- `syllabus-template.latex` - LaTeX template for the syllabus
- `syllabus.pdf` - Compiled syllabus document
- `README.md` - This file

### gh-pages Branch
Hosts the complete course website built with [Quarto](https://quarto.org). The website includes:
- Course overview and navigation
- Detailed syllabus information
- Lecture schedule with readings
- Assignment timeline and guidelines
- Interactive search functionality

## Website Technology

The website is built using Quarto, an open-source scientific and technical publishing system. The `_quarto.yml` configuration file defines the website structure, theme, and navigation. All generated website files are stored in the `docs/` folder, which allows for easy deployment on GitHub Pages.

## Building the Website

To build the website locally:

1. Install [Quarto](https://quarto.org/docs/getting-started/)
2. Navigate to the `gh-pages` branch
3. Run `quarto render` to generate the website
4. The built files will appear in the `docs/` directory

For more information on Quarto website development, see <https://quarto.org/docs/websites/>.

## Course Content

The course covers six main modules:

1. **Orientation** - Introduction to AI and course overview
2. **How AI systems are designed** - Dataset design, learning paradigms
3. **Language and perception** - Natural language processing and computer vision
4. **Retrieval, generation and pipelines** - Modern AI systems and workflows
5. **Data ethics and bias** - Ethical considerations and bias mitigation
6. **Policy, governance and social impact** - Regulation, privacy, and societal effects
